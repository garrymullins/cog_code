IBM Cognos Sample Programs

DimensionManagementServiceTest.java - a sample test program for the dimensionManagementService.

The Java DimensionManagementServiceTest sample program shows how to run dimensionManagementService requests in IBM Cognos.


Dependent Files

Associated files are contained under the java sample folder.
Additional files required for running are identified in the CLASSPATH definitions in run.bat file.
IBM Cognos dimensionManagementService must be installed and configured as explained in the IBM Cognos Business Viewpoint Install Guide. IBM Cognos Business Viewpoint Server must be started.

How To Run The Sample Program

Double click on the file run.bat. Connect to IBM Cognos.

A window will be opened containing:
    - IBM Cognos URL text box
    - Run Request command button
    - Clear command button
    - XML API Request input pane
    - XML API Response output pane

Open an XML file that contains a valid IBM Cognos Business Viewpoint request. The request must conform to the schema of the Business Viewpoint XML API. Click on the 'Run Request' button to run the XML API request. The test program will send the request to IBM Cognos dimensionManagementService. The service will forward the request to IBM Cognos Business Viewpoint Server.

Expected Results

The output pane will display the XML response returned by IBM Cognos Business Viewpoint Server. The response will conform to the schema of the Business Viewpoint XML API.

Documentation

The IBM Cognos Software Development Kit Developer Guide contains additional information about the samples. See the chapter titled 'Methods' and the appendix titled 'Code Samples and Language-Specific Coding Practices'.

Please note that the samples are not intended to be treated as end-user applications. Refer to the sample code for examples of how to use the API when developing your own applications.
