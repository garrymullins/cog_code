IBM Cognos Sample Programs

GroupsAndRolesGUI.java - Create, add and delete groups and roles.

The Java GroupsAndRolesGUI sample program provides methods to create, add, and delete groups and roles.


Dependent Files

Associated files are contained under the java sample folder.
Additional files required for running are identified in the CLASSPATH definitions in the file run.bat.

How To Run The Sample Program

Double click on the file run.bat.

A window will be opened containing:
    - Select Option dropdown box
    - Run Option command button
    - URL text box
    - Output pane
Select the desired sample option from the Select Option dropdown box and click on the 'Run Option' button to run the selected option.

Expected Results

When the CreateGroup option is selected, a group is created in the Cognos namespace.
When the CreateRole option is selected, a role is created in the Cognos namespace.
When the DeleteGroup option is selected, the specified group is deleted.
When the DeleteRole option is selected, the specified role is deleted.
When the Add User to Group option is selected, the specified user is added to the specified group.
When the Add User to Role option is selected, the specified user is added to the specified role.
When the Remove from Group option is selected, the specified user is removed from the specified group.
When the Remove from Role option is selected, the specified user is removed from the specified role.

A confirmation message is sent to the Output pane upon completion.

If you run samples that affect security settings, you may wish to use a test account. If you use a real account, you may need to ask your administrator to restore permissions or capabilities that you remove. If there is only one administrator account, do not run these samples as that user.
